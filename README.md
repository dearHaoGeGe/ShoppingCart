# ShoppingCart
##类似于饿了吗、百度外卖的点餐效果


* 左侧ListView可以联动右侧的ListView
* 右侧ListView分组可以悬停到顶端
* 添加产品有动画效果
* 购物车列表依赖于com.flipboard:bottomsheet-core:1.5.1
* 本demo参考[这里](https://github.com/fengyongge/shopcar.git),但是这个demo左右两个ListView没有设置关联和分组悬停。
